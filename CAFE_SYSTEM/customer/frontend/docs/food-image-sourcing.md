# Food Image Sourcing

Use this for menu items that still fall back to generic category images.

Recommended stock sources:
- Pexels license: https://help.pexels.com/hc/en-us/articles/360042295174-What-is-the-license-of-the-photos-and-videos-on-Pexels
- Pixabay license summary: https://pixabay.com/service/license-summary/
- Unsplash license: https://unsplash.com/license

Notes:
- Prefer images without visible brand logos, packaging, or watermarks.
- For items that are hard to find exactly, use the closest visual match rather than a misleading photo.
- If you use Unsplash, be extra careful with visible logos or branded cups: https://help.unsplash.com/en/articles/14224409-what-if-there-is-a-brand-logo-in-an-image-on-unsplash
